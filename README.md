## Getting Started

```
$ git clone git@github.com:mathdebate09/weather-app.git
$ cd weather-app
$ npm run build
$ npm start
```

## Learning Outcomes

- concepts of callbacks for async, promises, async-await
- how to take inputs from an api and serve them on the page
